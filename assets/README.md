# Assets

This directory contains images and media for the README.

## Required Assets

### 1. Logo (`vibe-check-logo.png`)

- Size: 120x120px
- Format: PNG with transparent background
- Style: Modern, clean, related to design/consistency

**Suggestions:**
- Checkmark with design elements
- Palette/color wheel with check
- Grid/layout icon with checkmark

### 2. Demo GIF (`demo.gif`)

- Width: 700px (max)
- Duration: 15-30 seconds
- Frame rate: 15fps (keep file size small)

**What to show:**
1. Run `vibe check ./src` — show issues detected
2. Run `vibe init ./src` — show setup completing
3. Open Claude Code — show MCP connected
4. Ask Claude to use design system — show it working

### Recording Tools

- [Gifox](https://gifox.app/) (macOS)
- [LICEcap](https://www.cockos.com/licecap/) (Windows/macOS)
- [Peek](https://github.com/phw/peek) (Linux)
- [asciinema](https://asciinema.org/) + [svg-term](https://github.com/marionebl/svg-term-cli) (terminal only)

### Tips for Good Demo GIF

1. Use a clean terminal theme
2. Increase font size for readability
3. Type at a reasonable pace (or use a script)
4. Pause briefly on important output
5. Keep total duration under 30 seconds
